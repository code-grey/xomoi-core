Engineering at the Edge

Xomoi-Core is engineered for industrial environments where memory is scarce, SD card IO is fragile, and network connectivity is highly intermittent. It entirely replaces the need for a complex control plane, external databases, or heavy container daemons.

By embedding Mochi-MQTT, a Zstandard-compressed SQLite Time-Series Database, and a Svelte UI directly into a single binary, Xomoi provides Kubernetes-level edge capability with a fraction of the computational footprint.

Lossless Ring Buffer

Xomoi achieves extreme throughput capabilities by decoupling network reads from disk IO. Packets are streamed into an in-memory Ring Buffer and asynchronously flushed to SQLite via bulk transactions.

Zstd TSDB & ULIDs

Traditional databases chew through SD card lifespan. Xomoi intercepts all telemetry, natively compresses JSON payloads using the Zstandard algorithm, and prevents timestamp collisions using mathematically sortable ULIDs.

Enterprise Escape Hatch

Utilizing the telemetry_pro.proto schema, Xomoi seamlessly handles multidimensional arrays (like LiDAR point clouds or AI tensors) via the BYTES_RAW field, acting as a high-speed, lossless data conduit.

Zero Dependency

A strictly sovereign ecosystem. No Node.js, no Docker, no external database, and no cloud tethers required for operation. Fully auditable and deployed in seconds.
