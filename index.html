Zero Dependency

A single, statically-linked binary. No Node.js, no external database, no microservice sprawl. Auditable by a single human.

HMAC-Lite Security

Bypasses heavy SSL/JWT buffers on 8-bit/32-bit hardware. Cryptographic integrity for the most constrained sensors.

Flash-Wear Protection

Optimized for SD-card longevity. High-frequency telemetry lives in RAM, with bulk snapshots to SQLite every 5 minutes.

Sovereign UI

Real-time Svelte 5 dashboard embedded via `go:embed`. Visualization and control that stays inside your local network.
